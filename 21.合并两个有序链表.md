### 题目
将两个有序链表合并为一个新的有序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 

```
示例：
输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4
```

### 解法
迭代，利用一个新建的节点依次将两条链表中最小的节点接上。
```js
/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} l1
 * @param {ListNode} l2
 * @return {ListNode}
 */
var mergeTwoLists = function(l1, l2) {
  let prevHead = new ListNode(-1);
  let prev = prevHead;

  // 当前 l1 或 l2 节点不能为空
  while(l1 !== null && l2 !== null) {
    // 如果当前 l1 小于 l2
    if (l1.val <= l2.val) {
      // 则接上 l1
      prev.next = l1;
      // 每次迭代移动指针
      l1 = l1.next;
    } else {
      // 否则接上 l2
      prev.next = l2;
      // 每次迭代移动指针
      l2 = l2.next;
    }

    // 每次迭代移动指针
    prev = prev.next;
  }

  // 最后如果有空节点和非空节点，适当处理
  prev.next = l1 === null ? l2 : l1;

  // 返回头结点
  return prevHead.next;
};
```
